## 🎨 О проекте

**Цветомузыка** — это приложение для Windows, которое анализирует звук с выхода звуковой карты в реальном времени и создает впечатляющее световое шоу из 8 прожекторов. Каждый прожектор реагирует на свой диапазон частот, создавая уникальную визуализацию музыки.

![Демонстрация работы](screenshot.png)

## ✨ Возможности

### 🎛️ Основные функции
- **Захват системного звука** — анализирует музыку, играющую на компьютере (через WASAPI Loopback)
- **8 независимых прожекторов** — каждый реагирует на свой диапазон частот
- **Настраиваемые частоты** — можно задать свой диапазон частот для каждого прожектора (20Hz - 20kHz)
- **Индивидуальные настройки** — чувствительность и минимальная яркость для каждого прожектора
- **Режимы расположения** — кругом, сеткой 2x4, 1x8 или 4x2
- **Выбор формы** — круглые или прямоугольные прожекторы
- **Регулировка размера** — от 50 до 1500 пикселей
- **Выбор цвета** — индивидуальный цвет для каждого прожектора (0-360° цветового круга)
- **Сохранение настроек** — все параметры сохраняются в JSON файл и восстанавливаются при запуске

### 🎯 Визуальные эффекты
- **Пульсация яркости** в такт музыке
- **Свечение и лучи** при высокой громкости
- **Частицы** разлетаются при битах
- **Спектрограмма** для визуализации частотного состава
- **Звездное небо** в фоне

### ⌨️ Управление
| Клавиша | Действие |
|---------|----------|
| `ESC` | Выход из программы |
| `H` | Скрыть/показать панель настроек |
| `S` | Скрыть/показать спектрограмму |
| `I` | Скрыть/показать текстовую информацию |
| `R` | Сброс частиц |

## 🚀 Установка

### Требования
- Windows 10/11
- [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)
- Звуковая карта с поддержкой WASAPI

Запустите ColorMusicVisualizer.exe от имени администратора

🎮 Использование
Первый запуск
Запустите программу от имени администратора (важно для захвата системного звука!)
Включите любую музыку или видео на компьютере
Наслаждайтесь цветомузыкой! 🎵

Настройка Stereo Mix (если нет звука)
Если программа не видит звук, включите Stereo Mix:
Нажмите правой кнопкой на иконку динамика в трее
Выберите "Звуки" → вкладка "Запись"
Нажмите правой кнопкой → "Показать отключенные устройства"
Найдите "Stereo Mix" или "Что слышно"
Включите его и сделайте устройством по умолчанию

Настройка параметров
Глобальные настройки
Общая чувствительность (0.5-8.0) — как сильно прожекторы реагируют на звук
Усиление яркости (0.5-5.0) — общая яркость свечения

Настройки отображения
Расположение — выбор схемы размещения прожекторов
Прямоугольные прожекторы — переключение формы
Размер — размер прожекторов (50-1500px)

Индивидуальные настройки прожекторов
Частота — диапазон частот (20Hz - 20kHz)
Цвет — оттенок прожектора (0-360°)
Чувствительность (0.1-5.0) — реакция на свой диапазон
Мин. яркость (0.05-0.5) — минимальное свечение в тишине

Сохранение профилей
Нажмите "СОХРАНИТЬ" — все настройки сохраняются в color_music_settings.json
Нажмите "ЗАГРУЗИТЬ" — восстановление сохраненных настроек
При закрытии программы настройки сохраняются автоматически

🛠️ Технические детали
Используемые технологии
.NET 8.0 — платформа
Windows Forms — графический интерфейс
NAudio — захват и анализ звука
WASAPI Loopback — захват системного звука
FFT (Fast Fourier Transform) — частотный анализ

🐛 Устранение неполадок
Проблема	Решение
Нет звука	Запустите от имени администратора, включите Stereo Mix
Ошибка инициализации аудио	Проверьте, что звуковая карта работает и установлен NAudio
Низкая чувствительность	Увеличьте "Общую чувствительность" до 5-8
Мерцание экрана	Используйте режим "Прямоугольные прожекторы"
Не сохраняются настройки	Проверьте права на запись в папке с программой
Высокая нагрузка на CPU	Уменьшите размер окна или отключите частицы
