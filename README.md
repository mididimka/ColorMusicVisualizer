## 📖 О проекте

**Color Music Visualizer** — это программа для Windows, которая в реальном времени анализирует звук с вашего компьютера и превращает его в захватывающее световое шоу. Восемь мощных прожекторов пульсируют и переливаются в такт музыке, создавая уникальную атмосферу.

![Демонстрация работы](screenshot.png)

## ✨ Особенности

### 🎛️ Основные возможности

| Функция | Описание |
|---------|----------|
| 🎵 **Захват системного звука** | Анализирует любую музыку, видео или игры на компьютере |
| 💡 **8 независимых прожекторов** | Каждый реагирует на свой диапазон частот |
| 🎨 **Настраиваемые цвета** | Индивидуальный цвет для каждого прожектора (0-360°) |
| 📊 **Частотные диапазоны** | Полная настройка границ от 20 Гц до 20 кГц |
| 🎚️ **Индивидуальная чувствительность** | Точная настройка реакции каждого прожектора |
| 📐 **Режимы расположения** | Круг, сетка 2×4, 1×8 или 4×2 |
| ⬛ **Форма прожекторов** | Круглые или прямоугольные |
| 🔍 **Регулировка размера** | От 50 до 1500 пикселей |
| 💾 **Сохранение настроек** | Все параметры сохраняются в JSON файл |
| 🚀 **Высокая производительность** | Оптимизированная графика с 60 FPS |

### 🎯 Визуальные эффекты

- ✨ **Пульсация яркости** в такт музыке
- 🌟 **Свечение и лучи** при высокой громкости
- 💫 **Частицы** разлетающиеся при битах
- 📈 **Спектрограмма** для визуализации частот
- ⭐ **Звездное небо** в фоне

### ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|

## 🚀 Установка

### Системные требования

- **ОС:** Windows 10 / 11 (64-bit)
- **Процессор:** 1.5 GHz или выше
- **ОЗУ:** 2 GB (рекомендуется 4 GB)
- **.NET Runtime:** 8.0 или выше
- **Звуковая карта:** Любая с поддержкой WASAPI

### Способ 1: Скачать готовый бинарник

1. Перейдите в раздел [Releases](ColorMusicVisualizer/Release)
2. Скачайте последнюю версию `ColorMusicVisualizer.zip`
3. Распакуйте в любую папку
4. Запустите `ColorMusicVisualizer.exe` **от имени администратора**

### Способ 2: Сборка из исходников

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/ColorMusicVisualizer.git
cd ColorMusicVisualizer

# Восстановите зависимости
dotnet restore

# Соберите проект
dotnet build -c Release

# Запустите
dotnet run

🎮 Использование
Первый запуск
Запустите программу от имени администратора — это важно для захвата системного звука!

Включите любую музыку, видео или игру

Наслаждайтесь цветомузыкой! 🎵

Настройка звука (если нет звука)
Если программа не видит звук, необходимо включить Stereo Mix в Windows:

Инструкция для Windows 10/11:
Нажмите правой кнопкой мыши на иконку 🔈 в системном трее

Выберите "Звуки" → вкладка "Запись"

Нажмите правой кнопкой в любом месте списка → "Показать отключенные устройства"

Найдите "Stereo Mix" или "Что слышно" (What U Hear)

Нажмите на него правой кнопкой → "Включить"

Снова нажмите правой кнопкой → "Использовать как устройство по умолчанию"

🐛 Устранение неполадок
Частые проблемы и решения
Проблема	Решение
❌ Нет звука	• Запустите программу от имени администратора
• Включите Stereo Mix в настройках Windows
• Проверьте, играет ли музыка на компьютере
• Перезагрузите компьютер
❌ Ошибка инициализации аудио	• Установите последнюю версию NAudio
• Проверьте драйвер звуковой карты
• Запустите программу от имени администратора
⚠️ Низкая чувствительность	• Увеличьте "Общую чувствительность" до 5-8
• Увеличьте громкость в системе
• Проверьте уровень звука в микшере
⚠️ Слабая яркость	• Увеличьте "Усиление яркости" до 4-5
• Увеличьте минимальную яркость прожекторов
• Проверьте яркость монитора
🐌 Низкий FPS	• Уменьшите размер окна
• Отключите частицы (клавиша R)
• Закройте другие ресурсоемкие приложения
• Используйте прямоугольные прожекторы
💾 Не сохраняются настройки	• Проверьте права на запись в папке программы
• Запустите от имени администратора
• Проверьте свободное место на диске
🎨 Пропадают цвета после загрузки	• Проверьте целостность файла настроек
• Используйте кнопку "Сброс" для восстановления
• Удалите файл настроек и перезапустите
