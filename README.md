## 📖 О проекте

**Color Music Visualizer** — это программа для Windows, которая в реальном времени анализирует звук с вашего компьютера и превращает его в захватывающее световое шоу. Восемь мощных прожекторов пульсируют и переливаются в такт музыке, создавая уникальную атмосферу.

![Демонстрация работы](screenshot.png)

## ✨ Особенности

### 🎛️ Основные возможности

| Функция | Описание |
|---------|----------|
| 🎵 **Захват системного звука** | Анализирует любую музыку, видео или игры на компьютере |
| 💡 **8 независимых прожекторов** | Каждый реагирует на свой диапазон частот |
| 🎨 **Настраиваемые цвета** | Индивидуальный цвет для каждого прожектора (0-360°) |
| 📊 **Частотные диапазоны** | Полная настройка границ от 20 Гц до 20 кГц |
| 🎚️ **Индивидуальная чувствительность** | Точная настройка реакции каждого прожектора |
| 📐 **Режимы расположения** | Круг, сетка 2×4, 1×8 или 4×2 |
| ⬛ **Форма прожекторов** | Круглые или прямоугольные |
| 🔍 **Регулировка размера** | От 50 до 1500 пикселей |
| 💾 **Сохранение настроек** | Все параметры сохраняются в JSON файл |
| 🚀 **Высокая производительность** | Оптимизированная графика с 60 FPS |

### 🎯 Визуальные эффекты

- ✨ **Пульсация яркости** в такт музыке
- 🌟 **Свечение и лучи** при высокой громкости
- 💫 **Частицы** разлетающиеся при битах
- 📈 **Спектрограмма** для визуализации частот
- ⭐ **Звездное небо** в фоне

### ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| <kbd>ESC</kbd> | Выход из программы |
| <kbd>H</kbd> | Скрыть/показать панель настроек |
| <kbd>S</kbd> | Скрыть/показать спектрограмму |
| <kbd>I</kbd> | Скрыть/показать информацию |
| <kbd>R</kbd> | Сброс частиц |

## 🚀 Установка

### Системные требования

- **ОС:** Windows 10 / 11 (64-bit)
- **Процессор:** 1.5 GHz или выше
- **ОЗУ:** 2 GB (рекомендуется 4 GB)
- **.NET Runtime:** 8.0 или выше
- **Звуковая карта:** Любая с поддержкой WASAPI

### Способ 1: Скачать готовый бинарник

1. Перейдите в раздел [Releases](https://github.com/yourusername/ColorMusicVisualizer/releases)
2. Скачайте последнюю версию `ColorMusicVisualizer.zip`
3. Распакуйте в любую папку
4. Запустите `ColorMusicVisualizer.exe` **от имени администратора**

### Способ 2: Сборка из исходников

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/ColorMusicVisualizer.git
cd ColorMusicVisualizer

# Восстановите зависимости
dotnet restore

# Соберите проект
dotnet build -c Release

# Запустите
dotnet run
